## управление

* Стрелочки двигают экран
* Мышкой меняется состояния клеток
* Клавишей пробел меняется состояние (закрашивание клеток и проигрывание самой жизни)
* Колесиком мышки можно менять масштаб
* Если в момент проигрывания закрасить клетку то вместе с ее закрашиванием программа автаматически перейдет в режим закрашивания клеток
* Флагами -w, -h можно менять размер поля
* флагом -s размер клетки в пикселях
* ./life -? выведет справку

## Вот здесь я расказываю, какой я умный:

Для реализации более понятного поведения интерфейса я использую паттерн `state`. Я реализовал состояния `UISetup` и `UIRun`. 
В первом игрок раставляет клеточки по полю, симуляция жизни в этот момент не активна. 
Второе проигрывает саму жизнь, позволяя человеку просто наблюдать.
Классы `UISetup` и `UIRun` являются *синглтонами Мейерса*

	UISetup* UISetup::get_instance() {
		static UISetup i;
		return &i;
	}
